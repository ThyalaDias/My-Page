Sobre o projeto

O projeto de Formulário para Login e Cadastro foi desenvolvido para fins de estudo e composição de portfolio. Sendo um dos projetos que mais gostei de elaborar, pude aplicar e testar as possibilidades tanto de design como de funcionalidade. Além de trabalhar com elementos básicos de HTML, pude me aventurar mais no CSS de forma mais complexa, onde tive que pensar nas configurações que se mostrassem mais interessantes e diferentes das que eu já havia feito, como acontece aqui, quando há a troca de tipos de formulário numa mesma página. Todas a composição das imagens do carrosel são de minha autoria, com execeção das imagens de pessoas que são do site Unplash.com. Aproveite e clique no carrosel para ver as artes! Aqui, usei HTML, CSS, Javascript e JQuery.
